xsd (XML Schema Definition, XML 结构定义)
spring-beans.xsd 说明

This defines a simple and consistent way of creating a namespace of JavaBeans objects, managed by a Spring BeanFactory, read by XmlBeanDefinitionReader (with DefaultBeanDefinitionDocumentReader).
Each “bean” element in this document defines a JavaBean. Typecally the bean class is specified, along with JavaBeans properties and/or constructor arguments.
A bean instance can be a “singleton” (shared instance) or a “prototype” (independent instance). Further scopes can be provided by extended bean factories, for example in a web environment et.al.
