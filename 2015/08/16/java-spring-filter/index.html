java-spring-filter

综述

Java Filter 用于对 HttpServletRequest 进行预处理与 HttpServletResponse 处理。可以用于请求与相应统计，用户权限的统一验证等操作。Filter 实现了 java.servlet.Filter 接口的服务端程序。主要用于对匹配的用户请求作统一处理。Filter 需要在 web.xml 中配置,随着 web application 启动而启动（只初始化一次)，只有在 web application 停止或重新部署时才销毁。
