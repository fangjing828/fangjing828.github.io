java-framework-config

配置在框架中的作用

  1. 避免硬编码。
  2. 指定映射关系。
  3. 系统运行时参数

    避免硬编码

    Spring 的 xml 配置文件(提供对象的初始参数)，配合 @Autowired 创建对象。

    指定映射关系

    Hibernate 的 ORM 映射,URL 与 Control 层的 Java 类的映射关系。

现状

要求：不丧失程序的灵活性，但要保持尽量简单。

形式：表现形式的多样化和命名约定的广泛使用性。

讨论

配置文件被认为是架起外界与程序之间的桥梁、是反应框架内部功能的一面镜子、更是程序是否灵活的一个重要标准
