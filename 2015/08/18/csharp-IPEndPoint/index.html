域名获取 IP 地址

域名绑定 IP

NOTE:BindIPEndPointDelegate is for choosing a local EndPoint for the request to originate from, not for choosing the remote one.

Q&A

  1. 异常处理:必须使用适当的属性或方法修改此标头

    通常通过 WebRequest.Headers 或 WebResponse.Headers 访问 WebHeaderCollection 类。
    某些公共标头被视为受限制的，它们或者直接由 API（如 Content-Type）公开，或者受到系统保护，不能被更改。

受限制的 Header

  • Accept
  • Connection
  • Content-Length
  • Content-Type
  • Date
  • Expect
  • Host
  • If-Modified-Since
  • Range
  • Referer
  • Transfer-Encoding
  • User-Agent
  • Proxy-Connection

修改受限制的 Header

  1. 继承 WebHeaderCollection，通过 AddWithoutValidate 方法来添加表头
  1. 判断 Header 的键值属性
foreach ( string key in headers.AllKeys)
{
    if (!WebHeaderCollection.IsRestricted(key))
        request.Headers.Add(key, headers[key]);
    else
    {
        request[key] = headers[key];
    }
}
