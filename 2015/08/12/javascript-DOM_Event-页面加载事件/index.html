综述

DOM 加载完成事件：DOMContentLoaded、window.onload、onreadystatechange,对于 IE 浏览器还可以使用 doscroll 方法进行判断。
DOMContentLoaded 事件是在 DOM 加载解析完毕时触发,是判断页面加载完成的首选方法。window.onload 需要等待所有的外部事件加载完毕时触发，方法被主流浏览器所支持。onreadystatechange 一般作为判断页面加载是否完成的备选事件。下面源码介绍如何利用页面加载方法实现 JQuery.ready。
